JavaScript is the core language of the web. It powers interactive websites, modern frontend frameworks, backend APIs, and even mobile and desktop applications.

This guide introduces JavaScript **from a developer's perspective**, focusing on what you actually need to build real projects.

---

## What is JavaScript?

JavaScript is a **high-level, dynamically typed programming language** that runs in the browser and on servers (via Node.js).

It allows you to:

- Manipulate the DOM
- Handle user interactions
- Communicate with APIs
- Build full-stack applications
